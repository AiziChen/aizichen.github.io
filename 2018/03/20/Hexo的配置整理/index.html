发表于 2018-03-20

Hexo 简介

Hexo是一个使用Mardown格式，使用github pages为服务基础，视觉可完全定制化的静态博客系统。

安装和使用Hexo

一、在github上创建自己的github pages

  1. 创建自己的github帐户，登录github主页：https://github.com 进行帐号注册;
  2. 进入github帐户，创建一个repository，在repository name中填写你注册的用户名 + .github.io，例如我的帐户名为AiziChen，那么我的repository name就是AiziChen.github.io；
  3. 接着点击底部的“创建 repository”，你的github pages就创建完成了
  4. 你的github目前的名字就是[用户名].gihub.io

二、下载并安装node.js和git版本管理软件

  1. 进入node.js的官网下载页：https://nodejs.org/en/download/ 进行选择对应的系统和版本下载，推荐下载LTS（长期支持版）；git进入git官网：https://git-scm.com/ 进行相应最新版本的下载；
  2. 安装node.js和git版本控制软件
  3. 将nodes.js和git的安装路径添加到系统的PATH环境变量中。
  4. 打开Mac/Linux的terminal，或者是Windows的cmd.exe命令行，输入git并回车，如果显示出git命令的使用方法则说明git的PATH环境变量设置成功；输入npm回车，如果显示出npm命令的使用方法则说明node.js的PATH环境变量设置成功；
