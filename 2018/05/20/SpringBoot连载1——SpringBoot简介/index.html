SpringBoot连载1——SpringBoot简介

发表于 2018-05-20 | 分类于 JavaWeb开发

简介

SpringBoot是一个JavaWeb应用的一站式解决方案，它其实只是一个工具箱，它里面含有了内置的Tomcat服务器，SpringMVC，Spring，以及其它的一些可选的功能。
它受Ruby On Rails框架的影响，主要是把“约定优于配置”和“快速开发”当成己任。因此使用它只需要非常少的配置文件。其实它使用的是Annotation对应用的组件进行配置。同时它还支持将所有的class和配置文件打包成一个独立的jar文件，使用java -jar webapp.jar就可以直接运行该Web应用。
SpringBoot这个Web工具箱不仅默认支持一些工具，还有如：
一.Servlet容器：
1.Tomcat（默认）
2.Jetty
3.Undertow

它还提供了一个spring-boot-devtools工具包来对SpringBoot应用进行热部署，使得开发者在不需要重启服务器的情况下即可对web应用进行实时调试。
SpringBoot还提供了一个命令行工具：Spring CLI (Spring Command Line Interface的缩写），它支持将单个groovy程序直接通过spring工具来进行Web程序的测试，目的是使得开发者在编写少量的源代码的情况下和不使用任何的配置文件的情况下就可以完美地运行一个Web程序，方便开发者们进行快速测试。

SpringBoot还提供了多元化的Annotation支持，使得开发者在几乎只使用一个Annotation就可以对所有的Router和Handler等进行全部的配置。

再者，SpringBoot推荐只使用Annotation和yml进行配置。
