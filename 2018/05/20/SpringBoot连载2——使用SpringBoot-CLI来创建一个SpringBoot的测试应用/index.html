SpringBoot连载2——使用SpringBoot CLI来创建一个SpringBoot的测试应用

发表于 2018-05-20 | 分类于 JavaWeb开发

一.安装Spring CLI

目前Spring CLI的最新版本为2.0，下载地址为：https://repo.spring.io/release/org/springframework/boot/spring-boot-cli/2.0.2.RELEASE/spring-boot-cli-2.0.2.RELEASE-bin.zip
下载并解压到某一目录下，然后在系统中为其设置PATH环境变量。它只有一个命令：spring，主要用来对groovy程序进行编译并运行成Web应用。
