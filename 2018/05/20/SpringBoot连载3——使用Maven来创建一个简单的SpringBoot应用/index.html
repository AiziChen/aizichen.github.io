SpringBoot连载3——使用Maven来创建一个简单的SpringBoot应用

发表于 2018-05-20 | 分类于 JavaWeb开发

俗话说一个简单的SpringBoot应用开发就是SpringBoot的精髓，因此这个“简单的SpringBoot应用”它并不“简单”，更不是没用的，恰恰是非常值得学习的。

使用Maven来创建一个简单的SpringBoot程序

零. 下载Maven插件。

首先需要到Apache官网下载maven插件，推荐使用3.3.9版本，它的下载地址为：http://www-us.apache.org/dist/maven/maven-3/3.3.9/binaries/apache-maven-3.3.9-bin.zip，下载完成后，添加它的bin目录到系统的PATH环境变量下。打开一个命令行，在上面输入:

mvn -v
如果看到版本信息，说明环境变量已经添加成功了。

一.配置SpringBoot的Maven依赖，并下载Maven的SpringBoot依赖包

  1. 配置SpringBoot的Maven依赖：随便在一个路径下新建一个first-springbootapp目录，再在该目录下新建一个pom.xml文件。打开该文件，复制以下文本进去。

打开命令行然后进入first-springbootapp目录，在命令行中输入：

mvn dependency:tree
该命令会自动下载以上已经配置的SpringBoot依赖包，并且显示出各个包之间的依赖，此时注意保持计算机处于联网状态，否则将会出现一系列错误，第二次即使已经联网了也无法下载了，这个Maven的一个Bug。
如果成功执行完成，就会显示SUCCESS的字样，并展示出它们的依赖关系，否则会显示FAIL。

运行该Web应用

打开命令行，进入first-springbootapp目录，执行以下命令进行这个web应用的启动：

mvn spring-boot:run
接着使用浏览器访问：http://localhost:8080
即可看到效果。

打包成jar包并使用java命令直接运行该Web应用

  1. 打包
    由于在pom.xml文件中定义了spring-boot-maven-plugin插件，因此我们可以使用Maven来直接将该Web应用打包成为一个jar包。
    打开命令行，进入first-springbootapp目录，使用如下的Maven命令进行对此Web应用的打包：

    mvn package
    接着你就会在target目录下发现一个名为”first-springbootapp-0.0.1-SNAPSHOT.jar”的jar包，该jar包其实就是一个完整的web应用，并可以使用java命令来直接运行。

  2. 运行该Web应用：

    java -jar target/first-springbootapp-0.0.1-SNAPSHOT.jar
    接着使用浏览器访问：http://localhost:8080
    即可看到效果。
